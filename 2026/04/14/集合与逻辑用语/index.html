集合和逻辑用语

本文转载自知乎：高中数学老师小易。

一、集合

1. 集合的概念

  • 把一些确定的、不同的对象看成一个整体，就是一个集合。
  • 集合中的每个对象叫做这个集合的元素。
  • 元素与集合的关系：
    • a∈Aa\in Aa∈A：aaa 是集合 AAA 的元素
    • a∉Aa\notin Aa∈/A：aaa 不是集合 AAA 的元素

2. 常用数集

  • N\mathbb NN：自然数集（含 0）
  • N+\mathbb N_+N+​ 或 N∗\mathbb N^*N∗：正整数集
  • Z\mathbb ZZ：整数集
  • Q\mathbb QQ：有理数集
  • R\mathbb RR：实数集
  • C\mathbb CC：复数集

3. 集合的表示方法

  1. 列举法：把元素一一列举，如 {1,2,3}\{1,2,3\}{1,2,3}
  2. 描述法：{x∣p(x)}\{x\mid p(x)\}{x∣p(x)}
  3. 图示法（Venn 图）：直观表示集合关系

4. 集合间的关系

  • 子集：若对任意 x∈Ax\in Ax∈A 都有 x∈Bx\in Bx∈B，则 A⊆BA\subseteq BA⊆B
  • 真子集：A⊆BA\subseteq BA⊆B 且 A≠BA\neq BA=B，记作 A⫋BA\subsetneqq BA⫋B
  • 相等：A⊆BA\subseteq BA⊆B 且 B⊆A  ⟺  A=BB\subseteq A\iff A=BB⊆A⟺A=B
  • 空集：不含任何元素，记作 ∅\varnothing∅
    • ∅\varnothing∅ 是任何集合的子集
    • ∅\varnothing∅ 是任何非空集合的真子集

5. 集合的运算

  1. 交集：A∩B={x∣x∈A 且 x∈B}A\cap B=\{x\mid x\in A \text{ 且 } x\in B\}A∩B={x∣x∈A 且 x∈B}
  2. 并集：A∪B={x∣x∈A 或 x∈B}A\cup B=\{x\mid x\in A \text{ 或 } x\in B\}A∪B={x∣x∈A 或 x∈B}
  3. 补集：∁UA={x∣x∈U 且 x∉A}\complement_U A=\{x\mid x\in U \text{ 且 } x\notin A\}∁U​A={x∣x∈U 且 x∈/A}

6. 重要公式与结论

  • A∩A=A, A∪A=AA\cap A=A,\ A\cup A=AA∩A=A, A∪A=A
  • A∩∅=∅, A∪∅=AA\cap\varnothing=\varnothing,\ A\cup\varnothing=AA∩∅=∅, A∪∅=A
  • A∩∁UA=∅A\cap\complement_U A=\varnothingA∩∁U​A=∅
  • A∪∁UA=UA\cup\complement_U A=UA∪∁U​A=U
  • ∁U(∁UA)=A\complement_U(\complement_U A)=A∁U​(∁U​A)=A
  • 交换律：A∩B=B∩A, A∪B=B∪AA\cap B=B\cap A,\ A\cup B=B\cup AA∩B=B∩A, A∪B=B∪A
  • 结合律、分配律略
  • 容斥原理：

    ∣A∪B∣=∣A∣+∣B∣−∣A∩B∣|A\cup B|=|A|+|B|-|A\cap B| ∣A∪B∣=∣A∣+∣B∣−∣A∩B∣

7. 子集个数

若集合 AAA 中有 nnn 个元素，则

  • 子集个数：2n2^n2n
  • 真子集个数：2n−12^n-12n−1
  • 非空子集：2n−12^n-12n−1
  • 非空真子集：2n−22^n-22n−2

二、常用逻辑用语

1. 命题

  • 可以判断真假的陈述句叫命题。
  • 真命题、假命题。

2. 四种命题

原命题：若 ppp，则 qqq

  • 逆命题：若 qqq，则 ppp
  • 否命题：若 ¬p\neg p¬p，则 ¬q\neg q¬q
  • 逆否命题：若 ¬q\neg q¬q，则 ¬p\neg p¬p

重要结论：
原命题   ⟺  \iff⟺ 逆否命题
逆命题   ⟺  \iff⟺ 否命题

3. 充分条件与必要条件

若 p⇒qp\Rightarrow qp⇒q：

  • ppp 是 qqq 的充分条件
  • qqq 是 ppp 的必要条件

若 p⇔qp\Leftrightarrow qp⇔q：

  • ppp 是 qqq 的充要条件

4. 含逻辑联结词的命题

  • p∧qp\land qp∧q（且）：全真才真
  • p∨qp\lor qp∨q（或）：一真即真
  • ¬p\neg p¬p（非）：真假相反

5. 全称量词与存在量词

  • 全称量词：∀\forall∀（任意）
  • 存在量词：∃\exists∃（存在）

命题的否定：

  • ∀x,p(x)\forall x,p(x)∀x,p(x) 的否定：∃x,¬p(x)\exists x,\neg p(x)∃x,¬p(x)
  • ∃x,p(x)\exists x,p(x)∃x,p(x) 的否定：∀x,¬p(x)\forall x,\neg p(x)∀x,¬p(x)

注意：
命题的否定只否定结论；
否命题既否定条件又否定结论。
